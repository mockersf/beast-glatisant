Github ❤ Playground

Integration between github issues and playground.

Fork me, back to app

Small helper to get code sample from Github issues and comment, and send them to the Playground to ask for Clippy's output.
By extracting all code blocks from markdown, and gists from Playground links. Those code samples are then sent to the Playground if they are Rust code (marked by 
```rust
in markdown).
This website use a cookie to remember your Github token, should you choose so.

Functionnalities
  • Check an issue: read everything from the issue specified. Your Github token is optional for this but you may encouter rate limit.
  • Check new issues and comments: read everything from the repo, and filter by date, by default for the last three days. A Github token is required and can be created from Github settings.
